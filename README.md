# 减肥打卡网页版小应用

这是一个纯前端的减肥打卡小工具，用于：

- 记录每日体重和备注；
- 展示体重变化趋势和简单统计分析；
- 根据每日变化给出相应的激励文案；
- 在浏览器中本地保存数据（不会上传到服务器）。

## 使用方式

1. 将本项目文件放在同一个文件夹（已包含：`index.html`、`styles.css`、`script.js`）。
2. 推荐使用任意静态服务器或 VS Code 的 Live Server 插件打开：
   - 方式一：直接用浏览器打开 `index.html`（双击或拖到浏览器里）。
   - 方式二：在该目录下使用静态服务器（例如：`npx serve .`），然后在浏览器访问提示的地址。
3. 在页面中：
   - 先在右上角设置“目标体重”（可选）并保存；
   - 在“今日打卡”区域选择日期和体重，填写备注（可选），点击“保存打卡”；
   - 下方可以查看统计信息、折线趋势图以及历史记录表格。

## 功能说明

- 支持 **多用户**：右上角切换或新建用户，每个用户独立保存身高、目标体重和记录。
- 支持 **身高** 与 **BMI**：输入身高后，统计卡会显示当前 BMI 与状态。
- 支持 **批量删除**：表格可勾选多条记录后批量删除，也可清空当前用户的全部记录。
- 数据保存在浏览器 `localStorage`（不会上传），新版本使用键 `weight_app_state_v2`，会自动从旧版数据迁移。
- 随时点击“清空全部数据”可删除当前用户所有记录；“删除所选”可批量删除选中的日期。
